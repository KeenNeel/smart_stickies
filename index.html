スマート付箋（Smart Stickies Ver1.00）仕様書

1. 概要

Smart Stickies は、
デスクトップ常駐型の高機能付箋アプリケーションです。

単なるメモ管理にとどまらず、

  • AIによる タスク分析・助言
  • 翻訳・校正・敬語変換
  • 画像生成
  • ガントチャートによる工程管理

これらの機能を一体化した


「考える・整理する・実行する」を一枚の付箋で完結させます

2. システム構成図

図1：システム全体構成（イメージ）

構成要素

  • デスクトップ全体を覆う Electron透明レイヤー
  • 複数の付箋ウィンドウ（期限状態により色変化）
  • 付箋内容をリスト表示しチャート表示するリスト画面
  • 画面下部に常駐する ドック（操作パネル）
  • ローカル・クラウドファイルストレージ
  • Google Gemini APIによタスク支援

データフロー

  • 付箋データ・設定 → ローカルJSONへ保存
  • AI処理要求 → Google Gemini API
  • 生成結果 → 付箋へ即時反映

5. 特殊機能：Neko Breakout（休憩モード）※K氏追加要件

概要 📌 「休憩＝離脱」ではなく「軽い没入」で集中力を回復

作業の合間に起動できる 遊び心のある集中回復モード。

  • 起動キー：Ctrl + Alt + C
  • ゲーム内容
  • 付箋 = ブロック
  • 猫型ボールでブロック崩し
  • 演出
  • 画面下部を横切る「喋る動物たち」
  • 仕事あるある / ジョーク / なぞなぞ
  • TTS（音声読み上げ）対応
  • コンボ数に応じてBGMが変化
  • カントリー / ロック / ジャズ 等
  • 特定スコアで Speed Upイベント 発生

（参考資料）動物のつぶやきはトピックスを選択可能 AIによって生成されたセリフがコード内に多数埋め込まれている
